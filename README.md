# Task Trove

Task Trove is my implementation of the classic project "ToDo" app but with much better theme overall and a lot of features which are properly refined.

## Tech Stack Used
- Flutter
- Dart 
- Flutter Riverpod 2.0
- SQLite
- Firebase
